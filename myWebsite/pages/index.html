About Webserv

Webserv is a fast, lightweight HTTP/1.1 server writtent in C++ that makes it easy to host static sites and run dynamic scripts. With simple configuration, you can serve files, handle uploads, and customize error pages. It's perfect for learning HTTP fundamentals or powering small web applications.
